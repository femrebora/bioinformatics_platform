Platform capabilities
Everything built-in
No third-party add-ons needed — every feature ships as first-class functionality.
🎯

Auto-wiring

Greedy bipartite port matching + topological DAG layout. Drop nodes and edges appear automatically.

⌨️

Keyboard-first UX

Ctrl+S save · Ctrl+Z/Y undo/redo · Cmd+K spotlight command palette with catalog search.

📋

8 Pipeline Templates

One-click gallery: RNA-seq, variant calling, ATAC-seq, metagenomics, long-read assembly, and more.

↩️

Undo / Redo

50-step snapshot stack. Covers drops, connections, loads, template applications, and new canvas.

📤

SRA / URL Input

Type an SRR/ERR/DRR accession or HTTPS/FTP URL — the worker downloads at runtime. No pre-staging needed.

📡

Live Log Streaming

Redis-backed per-job log buffer. Frontend polls every 1.5s, renders a dark terminal pane with auto-scroll.

🔐

Auth + Rate Limiting

JWT register/login · per-user job scoping · slowapi rate limits (5 reg / 10 login / 20 jobs per min).

💳

Stripe Payments

Cost-estimate modal before every run. Stripe-hosted checkout. Webhook triggers job creation atomically.

🔁

Cancel & Retry

Cancel revokes the Celery task (SIGTERM) and terminates the AWS Batch job. Retry clones the job instantly.

🗂️

Job History

Table of last 50 jobs with status, pipeline, tier, cost, and timestamp. Cancel / Retry buttons inline.

🔗

Paired-end Support

R1 + R2 upload slots with a separate file-out-r2 handle. Samplesheet generator fills fastq_2.

⬇

Presigned Downloads

Per-file S3 presigned GET URLs, verified against the job's result manifest. Copy JSON / export TSV.

Results panel
Six result renderers, one sliding panel
Auto-detected by result type. Slides in as a side drawer or expands to a centre modal with Summary / Data / Downloads / Raw tabs.
🧬

HLA Alleles

Locus-colour-coded allele cards. Class I / II grouping. Horizontal bar chart per gene. Copy-paste ready.

🔬

VCF Variants

Stat cards (total · PASS · SNP/INDEL). Donut chart. Sortable + filterable table. First 500 variants shown.

📉

Volcano Plot

SVG scatter for DE tables. Up / down / n.s. colouring. Significance & fold-change thresholds displayed.

📋

Count Matrix / Table

Generic sortable, searchable, paginated table. CSV & TSV auto-detected. Volcano rendered when DE columns present.

📄

HTML Report

MultiQC, Snakemake, or any HTML report. Summary card + "View Report" button that opens in a sandboxed iframe.

📦

File List

Emoji-icon per MIME type. File description, path, size. One-click presigned download or "Open" for HTML reports.
